Save & Play est une application qui permet de sauvegarder des fichiers audios et vidéos sur son Windows Phone.
Il est ainsi possible de les jouer ultérieurement sans avoir besoin d'être connecté à internet.

Différents moyens sont à votre disposition pour télécharger ces fichiers videos/audios :
- Naviguez sur internet et cliquez sur le media de votre choix
- Connectez vous sur votre compte SkyDrive et sélectionnez le media de votre choix
- Connectez vous sur votre compte Dropbox et sélectionnez le media de votre choix
- Scannez un QRcode contenant un lien ciblant un media
- Entrez manuellement un lien ciblant un media

De plus, Save & Play propose les fonctionnalités suivantes :
- Deux modes de transfert : lié à ou détaché de l'application
- Les fichiers de votre choix peuvent être protégés par un mot de passe
- Les fichiers peuvent être rénommés et/ou organisés
- Les fichiers audio peuvent être joués en arrière plan
